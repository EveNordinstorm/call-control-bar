# Call Control Bar

An accessible, physics-based animated in-call control bar.

## Features

- **Camera & microphone toggles** — animated diagonal strikethrough drawn with an SVG spring animation
- **Device picker menus** — per-device input/output selection for camera and mic
- **Settings menu** — springs open upward with staggered item fade-ins
- **End call animation** — phone icon rotates on click, then remaining buttons stagger-slide out
- **CSS tooltips** — instant and accessible
- **WCAG 2.2 AA** — `aria-pressed`, `aria-expanded`, `aria-label`, `role="group"`, `role="menu"` throughout

## Stack

- React 19 + TypeScript
- Motion (animations)
- Lucide React (icons)
- CSS Modules

## Run locally

npm install
npm run dev
